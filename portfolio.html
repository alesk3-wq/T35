O projeto

O que é a T35 Academy?

Um sistema web completo para gerenciar cursos e treinamentos internos, com controle de inscrições, presenças e emissão automática de certificados — tudo acessível pelo navegador ou instalado como aplicativo no celular.

Gestão de cursos

Admin cria e gerencia cursos com título, descrição, instrutor, datas, horário de início, vagas e local. Cursos encerrados bloqueiam novas inscrições automaticamente.

Fluxo de inscrições

Candidato se inscreve → instrutor ou admin aprova → candidato registra presença com código → status vira Concluído ou Falta.

Certificados digitais

Ao concluir um curso, o candidato acessa a área de Certificados e baixa o PDF gerado automaticamente no navegador — sem servidor, sem custo extra.

Notificações

Admin envia avisos para todos os usuários diretamente pela plataforma. Sino com badge exibe as mensagens não lidas em tempo real.

Relatórios

Painel admin com dados de inscrições e cursos: totais de concluídos, faltas, ocupação por vagas — com filtros de busca e status.

App instalável (PWA)

Funciona como aplicativo nativo no Android (instalação automática) e iOS (via Safari). Página de instalação com guia passo a passo por plataforma.
